# Preliminary Trademark Notes

Date of pass: **2026-04-06**

This is a preliminary naming and collision pass, **not legal advice** and not a substitute for a formal clearance search.

## Current Read

### Bare Metal / BareMetal

Avoid as the primary public mark.

Reasons:

- already widely used in computing as a descriptive term
- existing software and tech uses make it crowded
- hard to search cleanly
- likely weaker as a protectable brand than a more distinctive name

### RawIron

Current best candidate.

Why:

- stronger as a product identity than Bare Metal
- keeps the industrial/material feel
- initial quick search did not surface an obvious active game-engine/software brand conflict in the exact lane
- a historical **RAW IRON** filing exists, but it appears abandoned

## Editor Name Status

No final editor name has been chosen yet.

The first round of common forge/tool nouns was rejected because the names felt wrong and/or looked too crowded:

- Anvil
- Smithy
- Bloom
- Temper
- Ironworks

## Practical Rule

Use a more distinctive mark for the product name, then use forge/metal language in taglines, subsystem names, and product copy.

## Follow-Up Work

- deeper software/game/tool clearance pass for **RawIron**
- editor naming pass that actually matches the project's taste
- domain and handle review once the editor name is stronger
