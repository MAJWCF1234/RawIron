# RawIron

## Official Direction

- Product name: **RawIron**
- Positioning/tagline: **"a bare-metal game engine"**

## Naming Intent

RawIron is the public-facing product identity.

The phrase **bare metal** remains useful as positioning, mood, and technical attitude, but it should not be the primary mark. It is better used in marketing language than as the product name itself.

## Why This Fits

- sounds native and industrial
- feels like a real engine product rather than a prototype codename
- keeps the material/forge vibe without leaning on a generic computing term as the mark
- works for both runtime and tools

## Product Family

- **RawIron**: engine/platform umbrella
- **RawIron Editor**: current editor product label
- **RawIron Player**: standalone runtime/game executable
- **RawIron Tools**: importers, cookers, packers, and support utilities

## Brand Rules

- prefer **RawIron** in logos, repo naming, packages, and product references
- **Raw Iron** can still appear in prose if a more human reading is useful
- keep **bare-metal game engine** descriptive, not brand-primary
